## 常用Dos命令

```bash
*： #盘符切换
dir #查看当前目录下的所有文件
cd #切换目录 change directory
cd .. #返回上级
cls #清理终端 clear screen
exit #退出终端
ipconfig #查看电脑ip
md * #新建名为*的文件夹
rd * #移除名为*的文件夹
cd>*.* #创建名为*.*的文件
del #删除名为*.*的文件

#打开应用
    calc
    mspaint
    notepad
    
#ping 命令
    ping www.baidu.com
```
