# Java方法

- main方法

  ```java
  public static void main(String[] args){
      
  }
  ```



- 加法

  ```java
      public int add(int a,int b){
          return a+b;
      }
  ```



**一个方法只作一个功能**

类似其他语言的函数，是一段**用来完成特定功能的代码片段**

**方法包含一个方法头和一个方法体**

- 修饰符：可选
- 返回值类型
- 方法名：方法的实际名字，需要遵守命名规范
- 参数类型：像占位符，可选，也可以不含任何参数
  - 形式参数：用来定义作用的
  - 实际参数：实际调用传递给他的参数
- 方法体：方法里包含的具体语句，定义该方法的功能

- **如果方法中存在返回值，一定要通过 return 返回出去**

  ```java
  修饰符 返回值类型 方法名(参数类型 参数名){
      ...
      方法名
      ...
      return 返回值;
  }
  ```



## 方法的重载

- 在一个类中，有相同的函数名称，但形式参数不同的函数
- **规则**
  - 方法名称必须相同
  - 参数列表必须不同
    - 个数不同、类型不同、排列顺序不相同等
  - 返回值类型可以相同可以不相同
  - 返回类型不同不足以成为方法的重载

## 可变参数

- 在指定参数类型后加一个省略号 ...

- 一个方法中只能指定一个可变参数，必须是方法的最后一个参数，任何普通参数都必须在他之前声明



## 递归

- 递归头：什么时候**不调用自身的方法**。**如果没有头，将陷入死循环**

- 递归体：什么时候**需要调用自身方法**

- **能不用递归就不用递归**

- 小计算、基数小的情况使用递归

  ```java
      public static int f(int n){
  
          if (n==1){
              return 1;
          }else {
              return n*f(n-1);
          }
  ```
